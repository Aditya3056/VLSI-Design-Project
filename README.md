# 📌 Project Overview

This project focuses on the design and implementation of a digital VLSI system using Verilog HDL. The objective is to design, simulate, verify, and analyze the performance of digital circuits while following the standard VLSI design flow.

The project includes RTL design, functional verification, waveform analysis, synthesis, and performance evaluation.

---

# 🎯 Objectives

- Design digital circuits using Verilog HDL
- Perform RTL simulation
- Verify functional correctness
- Generate simulation waveforms
- Optimize area, timing, and power
- Understand the complete ASIC/FPGA design flow

# ▶️ How to Run

### Clone Repository

```bash
git clone https://github.com/yourusername/VLSI-Design-Project.git

cd VLSI-Design-Project
```

---

### Compile

```bash
iverilog -o output RTL/design.v Testbench/tb_design.v
```

---

### Run Simulation

```bash
vvp output
```

---

### View Waveform

```bash
gtkwave waveform.vcd
```
